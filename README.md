# E-Commerce-Revenue-and-Customer-Retention-Analysis
## 📊 E-Commerce Data Analysis Project

### 🔍 Problem Statement
Analyzed an e-commerce dataset to identify revenue drivers, customer retention patterns, and operational inefficiencies.

### 📁 Dataset
Olist Brazilian E-Commerce Dataset (Kaggle)

### 🛠 Tools Used
SQL | Python | Power BI | Excel

### 📈 Key Insights
- Top 20% customers generate 60% revenue
- Delivery delays impact customer satisfaction
- Certain categories are high revenue but low profit
